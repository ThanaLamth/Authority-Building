# NFTEnex Authority-Building Package

This folder contains a 10-article authority-building batch for an NFT and digital ownership specialist site.

Prepared on 2026-07-10.

## Contents

- `articles/`
  - 10 draft list and comparison articles
- `nftenex-topic-research-2026-07-10.md`
  - research summary, topic rationale, and current-source notes
- `nftenex-content-tracker-2026-07-10.md`
  - status tracker for drafting, sourcing, and edit readiness

## Editorial Rules Used

- The first H2 answers the title directly.
- The rest of each article follows a deductive chain:
  - answer
  - criteria
  - use-case split
  - option-by-option analysis
  - mistakes and risks
  - final recommendation
- Top/list pages are structured for `Article` + `ItemList` schema.
- Exact 2026 product details that still need a final verification pass are marked `[needs source]`.

## Naming

The folder is named `nftenex` to match the live tmux session and the NFT-focused site concept for this batch.
