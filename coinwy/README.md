# coinwy Authority-Building Package

This folder contains a 10-article authority-building batch for a crypto site focused on how-to, tools, wallets, exchanges, and strategies.

Prepared on 2026-07-10.

## Contents

- `articles/`
  - 10 draft list-style commercial and editorial articles
- `coinwy-topic-research-2026-07-10.md`
  - topic selection and prioritization note
- `coinwy-first-10-article-briefs-2026-07-10.md`
  - brief pack for all 10 articles
- `coinwy-content-tracker-2026-07-10.md`
  - production tracker and QA handoff

## Editorial rules used

- The first H2 answers the title directly.
- The sections below follow a deductive chain:
  direct answer -> methodology -> best-by-use-case -> comparison -> risk -> FAQ.
- Each article includes:
  - H1
  - slug
  - primary keyword
  - meta title
  - meta description
  - schema recommendation
  - internal-link targets
  - source checklist
- Pages are written as source-aware drafts. Any claim that still needs final publication-grade checking is marked `[needs source]`.

## Publish posture

- These drafts are designed to be stronger than generic affiliate listicles.
- They aim for a balanced community-hub tone rather than hard-sell copy.
- Risk, custody, and suitability tradeoffs are included on purpose.
