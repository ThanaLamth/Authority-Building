# Kanalcoin Media Plan

Date: 2026-07-10

This file defines the minimum visual package that should accompany the 10-article batch.

## Shared Media Rules

- Use one featured image per article with a clean editorial, not promotional, look.
- Prefer original tables, annotated screenshots, and simple charts over generic stock art.
- Any product screenshot should be captured on the publication day or within the same update window.
- Add alt text that describes the comparison or data shown, not only the object in the image.
