# AI x Crypto Authority-Building Package

Date: 2026-07-10
Folder: `aicrypto`

This package contains a complete 10-article authority-building batch for an AI x Crypto editorial site.

## Contents

- `articles/`
  - 10 full draft list articles
- `research/aicrypto-source-notes-2026-07-10.md`
  - current-source notes used to support the draft set
- `aicrypto-content-tracker-2026-07-10.md`
  - article-by-article production status

## Editorial Rules Used

- The first `H2` in every article answers the title directly.
- The remaining `H2`s follow a deductive chain:
  - direct answer
  - methodology
  - ranked list
  - evidence and context
  - implications
  - FAQ
- Every article includes:
  - slug
  - meta title
  - meta description
  - JSON-LD schema block
  - internal-link targets
  - source notes

## Notes

- These drafts are designed to be strong editorial starting points, not publication-ready legal or investment advice.
- Crypto rankings and product status can change quickly. Fast-moving claims are either phrased cautiously or called out in verification notes.
- Schema uses `https://your-site.com/` placeholders and should be updated during CMS implementation.
